"u" alias

After this tutorial, you will be able to run u help and other commands as if you're running util help. This is called an alias.

Run vim ~/.bashrc or run it with any editor you like.

Add alias u="util" at the end.

If you're having a hard time doing this in vim, here is a step-by-step guide:

1. Press the I key on your keyboard.

2. Hold the down arrow key on your keyboard until your at the bottom of the file.

3. Type alias u="util" or paste it in by using ctrl+shift+v.

Now press the Esc key on your keyboard, type :wq and press enter.

Now restart the terminal and try running u help or any other util commands.


"p" alias for plugins

After this tutorial, you will be able to run p <plugin name> as if you're running util p <plugin name>. This is called an alias.

Run vim ~/.bashrc or run it with any editor you like.

2. Hold the down arrow key on your keyboard until your at the bottom of the file.

3. Type alias u="util p" or paste it in by using ctrl+shift+v.

Now press the Esc key on your keyboard, type :wq and press enter.

Now restart the terminal and try running your favorite plugin using p <plugin name>.

Creating plugins

After this tutorial, you have created a plugin that shows the text you specified.

Make sure you have a plugins folder, if you don't, run util plugin-folder. You can check using util plugins.

Run util createplugin and give the name and description you like, I'll be doing:

ribkix@penguin:~$ util createplugin
Plugin name: HelloWorld
Plugin description: Says what you specified

Created util_plugins/HelloWorld, edit it using util editplugin HelloWorld <description | plugin> [editor]

Now, run the following commands:

util editplugin HelloWorld plugin vim

Now add the following code:

echo "$1"

echo shows the text you specified, now we add a $, which means variable and we add the argument number to it, when we use *:1 it means we are checking every argument from the first argument. More information on the echo command, More information on variables, More information on argument variables

If you're unfamiliar with vim, here's a step-by-step guide:

1. Press the I key on your keyboard.

2. Type echo "$1" or paste it in by using ctrl+shift+v (not recommended).

Now press the Esc key on your keyboard, type :wq and press enter.

After that, run the following commands:

cd ..
cd ..

You're done! Now run util p HelloWorld or whatever the name is of your plugin and you should see the following:

ribkix@penguin:~$ util p HelloWorld Hello everyone!
Hello everyone!

Uploading plugins

After this tutorial, you have uploaded your plugin to the official Util.sh plugin library.

Run the generateupload command, util generateupload <plugin>

This will guide you through the upload generation setup.

We will automatically make a download link for the plugin code. Don't worry.

Now go to https://github.com/ribkix/util-plugins/blob/master/README.md, press the pencil icon in the top right of the file, add your upload code to the bottom, scroll down and click "Propose Changes".
