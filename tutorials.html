"u" alias

After this tutorial, you will be able to run u help and other commands as if you're running util help. This is called an alias.

Run vim ~/.bashrc or run it with any editor you like.

Add alias u="util" at the end.

If you're having a hard time doing this in vim, here is a step-by-step guide:

1. Press the I key on your keyboard.

2. Hold the down arrow key on your keyboard until your at the bottom of the file.

3. Type alias u="util" or paste it in by using ctrl+shift+v.

Now restart the terminal and try running u help or any other util commands.


"p" alias for plugins

After this tutorial, you will be able to run p <plugin name> as if you're running util p <plugin name>. This is called an alias.

Run vim ~/.bashrc or run it with any editor you like.

Add alias p="util p" at the end.

If you're having a hard time doing this in vim, here is a step-by-step guide:

1. Press the I key on your keyboard.

2. Hold the down arrow key on your keyboard until your at the bottom of the file.

3. Type alias u="util p" or paste it in by using ctrl+shift+v.

Now restart the terminal and try running your favorite plugin using p <plugin name>.
