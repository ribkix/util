About

Util.sh is a linux script full of commands to make it easier for you to remember very complicated commands or turn multiple commands into one.

Util.sh gets daily updates, and you can leave suggestions under Issues.


Install

WARNING THIS IS ONLY FOR LINUX!!!

This will guide you through installing util.sh on your linux computer.

Install Util.sh

Make sure you're in the home folder, check by running pwd, if it shows /home/ and then your username, you're good:

ribkix@penguin:~$ pwd
/home/ribkix

Now install Util.sh by running the following commands:

wget https://raw.githubusercontent.com/ribkix/util/master/util.sh -O util.sh
cp util.sh util
ls -l util
chmod u+x util
ls -l util
chmod a+x util
ls -l util

If the wget command didn't work for you for some reason, try downloading Util.sh here by right clicking the page and pressing "Save as". (MAKE SURE TO SAVE IT IN THE HOME FOLDER)

Now, run the following commands, if you don't have a root user, run sudo passwd root.

su
mv util /usr/local/games/util

If you don't want a root or don't want to use it for some reason, run the following command:

sudo mv util /usr/local/games/util

Run Util.sh

To now run util type util and then the command you want to run. For a list of commands run util help.

Make sure to check out some tutorials.
