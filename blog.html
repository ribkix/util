2020.8.11

Made disk command human readable

Made it so it shows the capacity in bytes, kilobytes, megabytes, gigabytes, etc.

2020.8.10

Added path command

This command will show your current working directory.

util path - shows current path

Added up to date and need to update text to the getchangelog command

Now, when you run the getchangelog command, you get to see if your version is up to date or not.

Added blog command

This command will send you to the official Util.sh website's blog page.

util blog - goes to the official Util.sh blog

Added blogpost command

This command will send you to the official Util.sh website's blog page's post of your current version.

util blogpost - goes to the official Util.sh blog post of your current version

Bug Fixes

Fixed that when viewing a plugin it doesn't delete util_viewplugin_description.txt when it failed

2020.8.9

A bit of polishing

Added "argument" in help legend and added a missing dot to the error message of not having the plugin you specified in the p command.

Added editexe command

This command will edit an executable you specified in your favorite editor.

util editexe <executable> [editor] - edits plugin you specified with the editor you specified, if there is no editor specified it will use your default editor

Added editplugin command

This command will edit the plugin you specified's description or code with your favorite editor.

util editplugin <plugin> <part: description | plugin> [editor] - edits executable with the editor you specified, if you specified no editor it opens it in your default editor

With this, we added <argument: thing1 | thing2> - select one of these options to the help command's legend due to this command having that.

With this, we also updated the output of the createplugin command to show how to edit the plugin.

Added size command

This command will show the size of the file you specified.

util size <file name> - shows the size of the file you specified

Added view command

This command will show all files in your current directory or the directory you specified if specified with the optional options you specified.

util view [path] [options] - views the files of the current path or if specified the specified path with optional options

Added generateupload command

This command will generate the upload code for the plugin you specified.

util generateupload <plugin> - generates upload code for plugin

Added sp command

This command will run the plugin you specified under sudo if needed.

util sp <plugin> - run plugin under sudo

Bug fixes

In plugins there is a problem with arguments with spaces but now this is fixed.

2020.8.8

A bit of polishing

We removed all the dots in the help command since lots of commands had it but lots of commands didn't.

Added up to date and need to update text to the getversion command

Now, when you run the getversion command, you get to see if your version is up to date or not.

Added tutorials command

This command will send you to the tutorials page on the official Util.sh website.

util tutorials - goes to the official Util.sh tutorials

Added [S] to executable command

Now you can run s-executable which will run the executable command without root (sudo).

Added delexe command

This command will delete the executable you specified.

[S] util delexe <executable> - deletes executable (root required)

Added argument support in plugins

Now you can use arguments inside plugins, for example: util p plugin_name argument1 argument2

Added [S] to refresh command

Now you can run s-refresh which will run the refresh command without root (sudo).

Added [S] to getupdate command

Now you can run s-getupdate which will run the getupdate command without root (sudo).

Added find command

This command will find the text you specified in the file you specified.

util find <text> <file name> - finds text you specified in the file you specified

Added open command

This command will open the file you specified with the default program for that file extension.

util open <file name> - opens file with the default program for the file extension

Bug fixes

  • You now can't download plugins that don't exist
