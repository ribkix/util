2020.8.8

A bit of polishing

We removed all the dots in the help command since lots of commands had it but lots of commands didn't.

Added up to date and need to update text to the getversion command

Now, when you run the getversion command, you get to see if your version is up to date or not.

Added tutorials command

This command will send you to the tutorials page on the official Util.sh website.

util tutorials - goes to the official Util.sh tutorials

Added [S] to executable command

Now you can run s-executable which will run the executable command without root (sudo).

Added delexe command

This command will delete the executable you specified.

[S] util delexe <executable> - deletes executable (root required)

Added argument support in plugins

Now you can use arguments inside plugins, for example: util p plugin_name argument1 argument2

 Added [S]  to refresh command

Now you can run s-refresh which will run the refresh command without root (sudo).

Added [S]  to getupdate command

Now you can run s-getupdate which will run the getupdate command without root (sudo).

Bug fixes

  • You now can't download plugins that don't exist
